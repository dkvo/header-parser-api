API Project: Request Headers Parser

    User Direction:

  1. The API end point is GET https://headers-parse.herokuapp.com/api/whoami.
  2. This API simply returns information about the local machine that open the link to the API
  3. THese information include the computer's IP address, the language that the client is capable of understanding and information about the browser and the local computer.

Usage:

https://headers-parse.herokuapp.com/api/whoami

Example Output:

{ ipAddress: "192.168.1.233", language: "en-US,en;q=0.9,vi;q=0.8", software: "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_13_4) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/69.0.3497.100 Safari/537.36" }
